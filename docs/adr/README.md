## ADR 작성 규칙

### 상태
- `제안됨`: 검토 중
- `승인됨`: 적용됨
- `폐기됨`: 다른 결정으로 대체됨
- `대체됨`: 새 ADR로 대체됨 (대체 ADR 링크 포함)

### 템플릿

```markdown
# ADR-XXX: 제목

## 상태
제안됨 / 승인됨 / 폐기됨 / 대체됨

## 맥락
왜 이 결정이 필요한가?

## 결정
무엇을 결정했는가?

## 결과
이 결정으로 인한 영향은?

## 관련 파일
- 영향받는 파일 목록
```
